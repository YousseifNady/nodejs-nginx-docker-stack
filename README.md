# Dockerized Node.js Application

A sample Node.js application demonstrating Docker and Docker Compose with:

- Express.js
- MongoDB
- Redis
- PostgreSQL
- Nginx reverse proxy
- Development and Production configurations

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Redis
- PostgreSQL
- Docker
- Docker Compose
- Nginx

## Run Development

```bash
docker compose -f docker-compose-dev.yml up --build
